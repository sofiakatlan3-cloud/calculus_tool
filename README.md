# Calculus Tool

## О проекте
Calculus Tool — это простой учебный проект на Python, предназначенный для выполнения базовых математических операций.
Проект создан в рамках учебного задания и подходит для начинающих.
Может использоваться студентами для знакомства с работой репозиториев GitHub и запуском Python-проектов.

---

## Функциональность
- Сложение чисел
- Вычитание
- Умножение
- Деление (с проверкой деления на ноль)
- Возведение в степень

---

## Требования
- Python 3.8+
- Внешние библиотеки не требуются

---

## Установка
1. Клонируйте репозиторий:

```bash
git clone https://github.com/sofiakatlan3-cloud/calculus_tool.git
cd calculus_tool
```

2. Создайте виртуальное окружение:

```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

4. Установите зависимости (если есть):

```bash
pip install -r requirements.txt
```

---

## Запуск проекта

Для запуска проекта выполните команду:

```bash
python main.py
```

После запуска в консоли отобразятся примеры выполнения математических операций.

---

## Структура проекта

* `main.py` — основной файл с реализацией функций
* `README.md` — описание проекта и инструкции по запуску
* `.gitignore` — файл для исключения временных и служебных файлов
* `requirements.txt` — список зависимостей проекта
